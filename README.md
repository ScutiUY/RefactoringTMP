## 개요
TMP(Travel Manage Planner)는 여행 테마에 따른 여행지 자동 추천 플랫폼입니다. 사용자의 여행 관리와 커뮤니티 리뷰 기능으로 사용자와 다른 여행자들이 다녀온 여행을 함께 공유 할수 있는 기능을 제공합니다.
